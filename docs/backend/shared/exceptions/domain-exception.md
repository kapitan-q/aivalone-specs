# DomainException Specification

## Назначение

`DomainException` — базовое исключение для ошибок бизнес-логики и нарушений инвариантов в доменной модели Shared Context. Используется для сигнализации о недопустимых операциях, нарушениях бизнес-правил или логических ошибок, не связанных с валидацией данных.

## Область применения

- Нарушение бизнес-правил в доменных моделях
- Недопустимые операции над сущностями или агрегатами
- Ошибки, связанные с логикой предметной области

## Пример использования

- Бросается при попытке выполнить запрещённую операцию над агрегатом
- Может быть унаследовано для конкретных случаев (например, при нарушении бизнес-ограничений)
