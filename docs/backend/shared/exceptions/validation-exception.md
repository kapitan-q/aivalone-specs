# ValidationException Specification

## Назначение

`ValidationException` — базовое исключение для ошибок валидации. Используется для сигнализации о нарушении инвариантов, некорректных данных или бизнес-правил на границах домена.

## Область применения

- Проверка входных данных (например, идентификаторов, enum'ов, форматов)
- Нарушение инвариантов Value Object или Entity
- Валидация на уровне сервисов и моделей Shared

## Пример использования

- Бросается при попытке создать объект с невалидными параметрами
- Может быть унаследовано для конкретных случаев (например, InvalidUuidException, InvalidMessengerException)
