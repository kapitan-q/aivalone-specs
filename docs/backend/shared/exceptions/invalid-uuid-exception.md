# InvalidUuidException Specification

## Назначение


`InvalidUuidException` — специализированное исключение, наследник [ValidationException](./validation-exception.md). Выбрасывается в случае, если переданная строка **не является валидным UUID** или не может быть преобразована в `Uuid`.

Используется для защиты домена от некорректных идентификаторов.

## Область применения

* `Uuid` Value Object
* Любые другие идентификаторы, основанные на UUID

## Когда выбрасывается

* Передана пустая строка
* Переданная строка не соответствует формату UUID (v1–v7)
* Невозможно создать UUID через `ramsey/uuid`

## Инварианты

* Доменные идентификаторы **всегда валидны**
* В домене **никогда не существует** объекта с невалидным UUID
* Является частным случаем `ValidationException`
