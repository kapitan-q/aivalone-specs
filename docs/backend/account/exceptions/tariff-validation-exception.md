# TariffValidationException Specification

## Назначение

`TariffValidationException` — исключение, выбрасываемое при попытке присвоить пользователю некорректный тариф или список тарифов.

## Наследование

Наследник базового `ValidationException` из Shared Context.

## Область применения

* Процесс обновления тарифов пользователя
* Проверка корректности тарифных кодов

## Когда выбрасывается

* Передан несуществующий или недопустимый тарифный код
* Список тарифов содержит ошибки или невалидные значения

## Инварианты

* В системе допускаются только валидные тарифные коды
