# UserNotFoundException Specification

## Назначение

`UserNotFoundException` — исключение, выбрасываемое при попытке обращения к несуществующему пользователю в домене Account.

## Наследование

Наследник базового `DomainException` из Shared Context.

## Область применения

* Поиск пользователя по идентификатору (`UserId`)
* Операции над пользователем (добавление/удаление мессенджера, обновление тарифов)

## Когда выбрасывается

* Пользователь с указанным `UserId` не найден в системе
* Операция невозможна из-за отсутствия пользователя

## Инварианты

* В домене не допускаются операции над несуществующими пользователями
